# SoccerStat

Тестовое приложение для просмотра футбольных лиг, команд и календарей матчей.

Стек:
- React 19
- TypeScript
- Vite
- MUI
- TanStack Query
- React Router

Реализовано:
- страница лиг с поиском и клиентской пагинацией;
- страница команд с поиском и клиентской пагинацией;
- календарь лиги с хлебными крошками, фильтром по диапазону дат и пагинацией;
- календарь команды с хлебными крошками, фильтром по диапазону дат и пагинацией;
- локализация интерфейса на русский;
- перевод времени матчей из UTC в локальное время пользователя;
- обработка ошибок API и превышения лимита запросов.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Создать `.env.local` по образцу `.env.example`.

3. Запустить приложение:

```bash
npm run dev
```

4. Собрать production-версию:

```bash
npm run build
```

## Переменные окружения

Используется одна переменная:
- `FOOTBALL_DATA_API_URL` — базовый адрес backend-прокси, по умолчанию `https://linguics.pro/api/football-data`.

## Скрипты


## Замечания

- `football-data.org` имеет ограничения бесплатного тарифа. При исчерпании лимита приложение показывает понятное сообщение пользователю.
- Приложение работает через proxy-сервер `linguics.pro`, а не обращается к `football-data.org` напрямую из браузера.
- Токен `football-data.org` хранится на proxy-сервере `linguics.pro`, а не во фронтенде.
- Такой подход нужен по двум причинам: чтобы не раскрывать токен в клиентской сборке и чтобы обойти CORS-ограничения `football-data.org` для браузерных запросов.
